How to install

1. Close Maya if it isn't already

2. Run the batch file found at
\COVID\CharacterandAI\Tools\DCC\Maya\

3. Open maya if the batch has sucessuflly copied the files.

4. You should see a new shelf called ProjectTools. 
You can leave the tool there or move it to any shelf you want.
Keep in mind that if the tool has an update it should still work.

5. You'll notice there's no icon to the tool, rtight click the icon and select Edit Shelf
Under the tab Shelves, browse the Icon Name and select "scripts/fbxporterUI_Icon64.png"

6. Click on Save All Shelves

7. The tool has been installed!
You can run it by cliking on it once and access the about by clicking twice
